# evccMQTT

Modul zur Integration von evcc ([evcc.io](https://evcc.io)) in IP-Symcon.

## 1. Voraussetzungen

* mindestens IPS Version 8.1
* eine installierte Version von evcc

## 2. Enthaltene Module

* MQTT API
    * evcc Standort
    * evcc Ladepunkt
    * evcc PV Anlage
    * evcc Batterie
    * evcc Extern geregeltes Gerät
    * evcc Fahrzeug
